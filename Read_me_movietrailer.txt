Read Me Doc:

This project helps in creating a HTML page listing your favourite movies with details like 
poster image, title and on clicking further opens the trailer of respective movie.

Required :

Software Installations:

1.Python
2.TextWrangler (for MAC)

Steps to execute the program:

1. Download media.py, entertainment.py and fresh_tomatoes.py files listed in the folder.
2.Open entertainment.py and then run the project.

Details:

1.media.py consists of the class Movie where we have initialised the class with instance 
variables of our interests like movie title, youtube Url etc.,
2.Entertainment.py consists of instances(choice of our favourite movies) of class movie.
3.fresh_tomatoes.py module  consists of open_movie_page() function which takes the list of
movies and helps in creating the required HTML page output
